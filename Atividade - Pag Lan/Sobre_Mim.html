Sobre Mim

Para criar este site, foram utilizadas linguagens de programação como HTML e CSS. O cabeçalho possui uma barra de navegação com links para as páginas Sobre Mim, Projetos e Contato. A cor de fundo foi personalizada com um gradiente de tons de rosa e roxo. A página apresenta uma imagem centralizada, redonda e com borda rosa, além de uma imagem do Projeto React com um texto descritivo acima. Também foi adicionado um rodapé com links para as redes sociais. Foram adicionados comentários para ajudar na compreensão do código. Esperamos que este exemplo seja útil para quem está aprendendo a criar sites com HTML e CSS. vel lobortis fringilla.
